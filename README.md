# 투석환자 일정 체크 도우미

투석환자별 검사일, 처방일, 혈관외과 방문일, 혈액검사 특이 소견, 비고를 간단히 메모하고 오늘 확인할 항목과 이번 달 일정을 한눈에 보는 Streamlit 웹앱입니다.

## 중요한 안내

- 이 앱은 일정 메모와 확인용 도구입니다.
- 의료 판단, 진단, 처방 추천, 처방 변경, 검사 결과 해석을 하지 않습니다.
- 병원 차트나 EMR과 직접 연동하지 않습니다.
- 실제 사용 시 환자 전체 이름, 주민번호, 연락처, 자세한 진단명 등 민감정보 입력을 피하고 병원 내부 기준에 맞는 최소 정보만 입력하세요.
- 현재 버전은 외부 API 없이 작동합니다.
- 현재 버전은 `현재 일정 저장하기`를 누르거나 일정을 추가하면 앱의 저장 파일에 일정을 저장합니다.
- 다음 접속 때 `저장된/과거 일정 불러오기`로 다시 불러올 수 있습니다.

## 설치 방법

```bash
pip install -r requirements.txt
```

## 실행 방법

```bash
streamlit run app.py --server.port 8501 --server.address 127.0.0.1
```

실행 후 브라우저에서 아래 주소를 엽니다.

```text
http://127.0.0.1:8501
```

## 사용 방법

1. 환자 이름 대신 `김OO`, `3병동 A-12`처럼 최소 식별 정보를 입력합니다.
2. 화면의 `작성` 탭에서 병동, 검사일, 처방일, 메모를 입력합니다.
3. `일정 추가` 버튼을 누르면 전체 환자 일정표에 추가되고 자동 저장됩니다.
4. `오늘 확인`, `가까운 일정`, `이번 달`, `전체 일정표` 탭을 눌러 필요한 화면만 따로 확인합니다.
5. `현재 일정 저장하기` 버튼으로 현재 표를 다시 저장할 수 있습니다.
6. 다음에 접속했을 때 `저장된/과거 일정 불러오기` 버튼을 누르면 저장했던 일정을 다시 불러옵니다.
7. `테스트용 샘플 보기` 버튼은 실제 과거 데이터가 없을 때만 테스트용으로 사용하세요.
8. 전체 일정은 `전체 일정표` 탭에서 CSV 또는 TXT 요약으로 다운로드할 수 있습니다.

## 샘플 입력 예시

- 환자 식별 정보: `김OO`
- 병동 또는 구분: `월수금 오전`
- 신환 검사일: 오늘 날짜
- 이번달 처방 날짜: 오늘 날짜
- 혈액검사 특이 소견: `칼륨 재확인 필요 메모`
- 비고: `오전 회진 전 확인`
- 확인 상태: `확인 전`

## 파일 업로드 준비 방법

CSV 또는 Excel 파일을 업로드할 수 있습니다. 가능하면 아래 열 이름을 사용하세요.

- 환자 식별 정보
- 병동 또는 구분
- 신환 검사일
- 1개월 검사일
- 3개월 검사일
- 6개월 검사일
- 이번달 처방 날짜
- 다음 처방 날짜
- 혈관외과 방문일
- 혈액검사 특이 소견
- 비고
- 확인 상태

일부 열이 없어도 앱이 빈 값으로 채우지만, 날짜 열 이름은 위 형식과 맞추는 것이 가장 안정적입니다.

## 로고 이미지 교체 방법

프로젝트 폴더 안에 `logo.png`, `brand.jpg`, `icon.webp`, `로고.png`처럼 이름에 `logo`, `brand`, `icon`, `로고`가 들어간 이미지 파일을 넣으면 앱 상단에 표시됩니다.

## 자주 발생할 수 있는 문제

- 앱이 열리지 않으면 실행 명령을 다시 확인하고 `http://127.0.0.1:8501`로 접속하세요.
- 업로드 파일이 읽히지 않으면 CSV 또는 Excel 파일인지 확인하고, 열 이름을 README의 목록과 맞춰 보세요.
- 일정이 보이지 않으면 먼저 `저장된/과거 일정 불러오기` 버튼을 눌러 보세요.
- 앱 배포 환경이 재시작되어 저장 파일이 초기화되는 경우를 대비해, 중요한 일정은 CSV 다운로드 파일도 함께 보관해 주세요.
- 다운로드 파일에는 사용자가 입력한 내용이 그대로 들어가므로 민감정보 포함 여부를 꼭 확인하세요.

## 나중에 추가하면 좋은 기능

- 장기 저장용 데이터베이스
- 로그인과 사용자별 권한
- 병원 내부 시스템 또는 EMR 연동
- 여러 사용자 동시 편집
- 월간 캘린더 보기
- 실제 알림, 문자, 카카오톡, 이메일, Google Calendar, Google Sheets 연동
- 병동별 권한 관리와 관리자 화면
- 기능이 커지면 `app.py`를 여러 파일로 나누어 정리
